Discord Pokémon Battle Bot

Discord Pokémon Battle Bot adalah game berbasis bot Discord yang memungkinkan pemain mendapatkan Pokémon secara acak dari PokeAPI dan bertarung melawan pemain lain. Bot ini dibuat menggunakan Python dengan konsep Object-Oriented Programming (OOP) dan asynchronous programming.

Fitur Utama

-Mendapatkan Pokémon secara acak melalui perintah !go
-Menampilkan informasi dan gambar Pokémon dengan !info
-Sistem pertarungan antar pemain dengan !attack @username
-Sistem pemberian makan dengan cooldown waktu menggunakan !feed

Cara menjalankan:
-Masukan token bot pada file config
-install discord
-jalankan main.py 
-siap untuk dimainkan
